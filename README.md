# 💼 Portfólio Pessoal — Felipe Campos Borges

Projeto de portfólio pessoal desenvolvido com foco em **design moderno, responsividade e experiência do usuário (UI/UX)**.  
A página apresenta informações profissionais, habilidades técnicas, galeria interativa e canais de contato.

## 📌 Sobre o projeto

Este projeto foi desenvolvido para apresentar meu perfil profissional como **Desenvolvedor Front-End / Full Stack**, destacando minhas habilidades, experiência e formas de contato de maneira visual e interativa.

A proposta foi criar uma landing page moderna com:

- Layout responsivo
- Interface elegante e profissional
- Navegação suave entre seções
- Carrossel de imagens
- Menu mobile
- Animações de entrada ao scroll
- Área de contato com links diretos

---

## 🛠️ Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **Google Fonts**
- **SVG Icons**

---

## 🎨 Funcionalidades

✅ Design moderno e responsivo  
✅ Menu mobile interativo  
✅ Carrossel de imagens automático e manual  
✅ Navegação por seções (scroll)  
✅ Animações com `IntersectionObserver`  
✅ Header dinâmico ao scroll  
✅ Botões de contato com links externos  
✅ Layout otimizado para desktop e mobile
